Check cost threshold

Thresholds for procurements are revised each January and can be found at https://www.gov.uk/government/publications/procurement-act-2023-guidance-documents-define-phase/guidance-thresholds-html (opens in new tab). If the procurement framework in use exceeds the threshold then it will need additional checks from the CAB (Commercial Assurance Board)

Use the threshold for the year when the framework or deal was created. If you are using a Bespoke procurement then use this years threshold.
