Check cost threshold

Thresholds for procurements are revised each January and can be found on the GOV.UK website (opens in new tab). If the procurement framework in use is above the threshold then it will need additional checks from the Commercial Assurance Board (CAB).

Use the threshold for the year when the framework or deal was created. If you are using a Bespoke procurement then use this year's threshold.
